# NostalgiApp Feedback

Welcome to the official feedback repository for [NostalgiApp](https://www.nostalgi.app)!

## How to Use This Repository

### Report a Bug
Found something broken? [Open a bug report](https://github.com/jupe69/nostalgiapp-feedback/issues/new?template=bug_report.yml)

Before reporting, please:
- Search [existing issues](https://github.com/jupe69/nostalgiapp-feedback/issues) to avoid duplicates
- Include steps to reproduce the bug
- Mention your browser and operating system

### Request a Feature
Have an idea to make NostalgiApp better? [Submit a feature request](https://github.com/jupe69/nostalgiapp-feedback/issues/new?template=feature_request.yml)

### General Discussion
For general questions and community chat, join our Discord (coming soon).

---

Thank you for helping improve NostalgiApp!
